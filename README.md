# Property Management Analytics Dashboard and Reporting Discord Bot

Professional bot for Property Management professionals.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
